torch_pharma.utils.math.inflate_batch_array(array, target)

Reshape a per-graph [batch_size] tensor to broadcast against a per-node or per-edge tensor of arbitrary rank. Appends singleton dimensions until array.ndim == target.ndim.

Parameters

  • array (Tensor) – 1D tensor of shape (B,), one value per graph in the batch.
  • target (Tensor) – Target tensor whose rank determines how many singleton dims to append.

Return type

Tensor — shape (B, 1, 1, ...) broadcastable against target

Example

from torch_pharma.utils.math import inflate_batch_array

gamma_t = torch.randn(32)            # one gamma per graph
x       = torch.randn(320, 3)        # 10 atoms * 32 graphs

alpha_t = inflate_batch_array(gamma_t, x)   # (32, 1)
z_t = alpha_t * x                            # broadcasts correctly
torch_pharma.utils.math.get_grad_norm(parameters, norm_type=2.0)

Compute the total gradient L_p norm across all parameters that have a gradient. Mirrors torch.nn.utils.clip_grad_norm_ but returns the value without clipping.

Parameters

  • parameters (Iterable[Parameter]) – Model parameters, typically model.parameters().
  • norm_type (float) – Exponent p for the L_p norm. Default: 2.0.

Return type

Tensor — scalar gradient norm

torch_pharma.utils.math.safe_norm(x, dim=-1, eps=1e-8, keepdim=False, sqrt=True)

Numerically stable L2 norm. Returns √(∑x² + ε) + ε when sqrt=True, preventing NaN gradients at zero vectors.

Parameters

  • x (Tensor) – Input tensor.
  • dim (int) – Reduction dimension. Default: -1.
  • eps (float) – Stability constant. Default: 1e-8.
  • keepdim (bool) – Keep reduced dimension. Default: False.
  • sqrt (bool) – Return L2 norm (True) or squared + eps (False). Default: True.

Return type

Tensor

torch_pharma.utils.io.save_xyz_file(path, positions, one_hot, dataset_info, id_from=0, name='molecule', node_mask=None)

Write a batched set of generated molecules to individual .xyz files. Decodes one-hot atom types using dataset_info['atom_decoder'].

Parameters

  • path (str or Path) – Output directory. Created if it does not exist.
  • positions (Tensor) – Atom coordinates, shape (B, N_max, 3).
  • one_hot (Tensor) – One-hot atom types, shape (B, N_max, num_atom_types).
  • dataset_info (dict) – Must contain 'atom_decoder' mapping index → element symbol.
  • id_from (int) – Starting index used in output filenames. Default: 0.
  • name (str) – Filename prefix. Default: 'molecule'.
  • node_mask (Tensor, optional) – Boolean mask shape (B, N_max) identifying real atoms. Default: None.

Return type

None

torch_pharma.utils.io.num_nodes_to_batch_index(num_samples, num_nodes, device)

Construct a batch_index tensor (e.g. [0,0,0,1,1,...]) from per-graph node counts. Used to initialise sampling without a data batch.

Parameters

  • num_samples (int) – Total number of graphs in the batch.
  • num_nodes (Tensor) – 1D long tensor of shape (num_samples,), per-graph atom count.
  • device (torch.device) – Target device for the output tensor.

Return type

Tensor — batch_index of shape (sum(num_nodes),)
