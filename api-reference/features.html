class torch_pharma.features.ScalarVector(scalar: Tensor, vector: Tensor)

A named tuple subclass for holding paired scalar and vector representations. The scalar channel has shape [N, s] and the vector channel has shape [N, v, 3]. Arithmetic operators (+, *) are overloaded to apply to both channels simultaneously.

Parameters

  • scalar (Tensor) – Scalar features, shape (N, s).
  • vector (Tensor) – Vector features, shape (N, v, 3).

Example

from torch_pharma.features import ScalarVector

sv = ScalarVector(scalar=torch.zeros(10, 64), vector=torch.zeros(10, 4, 3))

# Arithmetic
sv2 = sv + sv                         # elementwise add on both channels
sv3 = sv * 2.0                        # scalar multiply on both channels

# Indexing
sub = sv.idx(torch.tensor([0, 1, 2])) # selects rows 0,1,2 from both channels

# Flatten / recover
flat = sv.flatten()                   # → Tensor (10, 64 + 4*3)
sv4  = ScalarVector.recover(flat, vector_dim=4)  # → ScalarVector

# Masking
mask = torch.ones(10, dtype=torch.bool)
mask[5:] = False
sv_masked = sv.mask(mask)             # zeros rows 5-9 in both channels

torch_pharma.features.centralize(batch, key, batch_index, node_mask=None, edm=False)

Subtract the centre of gravity (CoG) of each graph from the attribute batch.key. Returns both the per-graph centroid and the centered attribute. Required for translation-invariant diffusion.

Parameters

  • batch (torch_geometric.data.Batch) – Molecular batch; must have attribute key.
  • key (str) – Attribute name to centralize (e.g. 'x' for positions).
  • batch_index (Tensor) – Graph membership index, shape (N,).
  • node_mask (Tensor, optional) – Boolean mask; if provided, masked nodes are excluded from the CoG calculation. Default: None.
  • edm (bool) – If True, use the EDM-style mask-weighted mean and assert masked values are negligibly small. Default: False.

Return type

Tuple[Tensor, Tensor] — (centroid shape (B, 3), centered_coords shape (N, 3))

torch_pharma.features.localize(pos, edge_index, norm_x_diff=True)

Construct a per-edge orthonormal local SE(3) frame f_ij. Each frame is a (3, 3) matrix whose columns are the direction vector, cross-product vector, and vertical vector derived from the positions of the two endpoint atoms.

Parameters

  • pos (Tensor) – Node positions, shape (N, 3).
  • edge_index (Tensor) – Edge index, shape (2, E).
  • norm_x_diff (bool) – Normalize difference and cross vectors to unit length (with +1 denominator to avoid division by zero). Default: True.

Return type

Tensor — local frames f_ij, shape (E, 3, 3)

Example

from torch_pharma.features import localize

f_ij = localize(batch.x, batch.edge_index)   # (E, 3, 3)
# columns: [x_diff, x_cross, x_vertical]
torch_pharma.features.scalarize(vectors, edge_index, frames, node_inputs=False, dim_size=None, node_mask=None)

Project equivariant vector features into rotation-invariant scalar features using local SE(3) frames. Computes frames @ vectors for each edge and concatenates the 3×3 inner products into a 9-dim scalar per entity.

Parameters

  • vectors (Tensor) – Vector features to scalarize, shape (E, 3, k) for edges or (N, 3, k) for nodes.
  • edge_index (Tensor) – Edge index, shape (2, E).
  • frames (Tensor) – Local frames returned by localize(), shape (E, 3, 3).
  • node_inputs (bool) – If True, scatter-aggregate edge scalarizations onto nodes. Default: False.
  • dim_size (int, optional) – Number of nodes N, required when node_inputs=True. Default: None.
  • node_mask (Tensor, optional) – Boolean node mask. Default: None.

Return type

Tensor — rotation-invariant scalars, shape (E or N, 9)

torch_pharma.features.vectorize(scalars, edge_index, frames, node_inputs=False, dim_size=None, node_mask=None)

Lift scalar gate values back into equivariant 3D vectors using local SE(3) frames. Interprets each group of 3 scalars as coefficients for the 3 frame axes, reconstructing direction-aware 3D vectors.

Parameters

  • scalars (Tensor) – Gate scalar features, shape (E or N, 3k) where k = number of frame triplets.
  • edge_index (Tensor) – Edge index, shape (2, E).
  • frames (Tensor) – Local frames, shape (E, 3, 3).
  • node_inputs (bool) – If True, scatter-aggregate results back to nodes. Default: False.

Return type

Tensor — equivariant vectors, shape (E or N, 3, k)

torch_pharma.features.safe_norm(x, dim=-1, eps=1e-8, keepdim=False, sqrt=True)

Numerically stable L2 norm. Computes √(sum(x², dim) + eps) + eps when sqrt=True, or sum(x², dim) + eps when sqrt=False. The double eps prevents NaN gradients at zero vectors.

Parameters

  • x (Tensor) – Input tensor.
  • dim (int) – Dimension along which to compute the norm. Default: -1.
  • eps (float) – Small constant for numerical stability. Default: 1e-8.
  • keepdim (bool) – Whether to retain the reduced dimension. Default: False.
  • sqrt (bool) – If True return L2 norm; if False return squared L2 + eps. Default: True.

Return type

Tensor — same shape as x except along dim

torch_pharma.features.get_nonlinearity(name, return_functional=False, slope=0.01)

Instantiate a nonlinearity by name string. Returns either an nn.Module instance or the corresponding functional for use inside forward().

Parameters

  • name (str or None) – One of: null/None → Identity, relu, leakyrelu, selu, silu/swish, sigmoid, tanh.
  • return_functional (bool) – If True, return F.relu / F.silu etc. instead of an nn.Module. Default: False.
  • slope (float) – Negative slope for leakyrelu. Default: 0.01.

Return type

nn.Module or callable

Example

from torch_pharma.features import get_nonlinearity

act = get_nonlinearity("silu")                          # nn.SiLU()
fn  = get_nonlinearity("relu", return_functional=True)  # F.relu
identity = get_nonlinearity(None)                       # nn.Identity()
