class torch_pharma.models.dynamics.egnn.EGNN_Sparse(feats_dim, pos_dim=3, m_dim=16, fourier_features=0, soft_edge=0, norm_coors=True, coors_tanh=True, update_feats=True, update_coors=True, coor_weights_clamp_value=None, dropout=0.0, **kwargs)

A single sparse EGNN message-passing layer (inherits from torch_geometric.nn.MessagePassing). For each edge (i,j) it computes a message m_ij from the squared distance and concatenated node features, then updates coordinates via a weighted sum of relative position vectors and updates node features via an aggregated MLP.

Parameters

  • feats_dim (int) – Dimensionality of input node features.
  • pos_dim (int) – Spatial dimensionality (3 for molecules). Default: 3.
  • m_dim (int) – Hidden dimension of the edge MLP. Default: 16.
  • fourier_features (int) – Number of Fourier distance encodings appended to edge features. 0 = raw squared distance only. Default: 0.
  • soft_edge (int) – When > 0, adds a sigmoid-gated attention weight on coordinate updates. Default: 0.
  • norm_coors (bool) – Apply CoorsNorm to relative position vectors before aggregation. Default: True.
  • coors_tanh (bool) – Apply tanh to the coordinate MLP output. Requires norm_coors=True. Default: True.
  • update_feats (bool) – Whether to update node features in the forward pass. Default: True.
  • update_coors (bool) – Whether to update node coordinates in the forward pass. Default: True.
  • coor_weights_clamp_value (float, optional) – If set, clamps coordinate update weights to this absolute value for numerical stability. Default: None.
  • dropout (float) – Dropout probability inside edge and node MLPs. Default: 0.0.

Example

from torch_pharma.models.dynamics.egnn import EGNN_Sparse

layer = EGNN_Sparse(feats_dim=64, pos_dim=3, m_dim=16)
# feats: (N, feats_dim), coors: (N, 3)
out_feats, out_coors = layer(feats, coors, edges, batch)  # edges: (2, E)

Return type Tuple[Tensor, Tensor] — (updated_feats, updated_coors)

class torch_pharma.models.dynamics.egnn.EGNN_Sparse_Network(n_layers, feats_dim, pos_dim=3, m_dim=16, fourier_features=0, soft_edge=0, norm_coors=True, coors_tanh=True, dropout=0.0, embedding_nums=None, embedding_dims=None, edge_embedding_nums=None, edge_embedding_dims=None, recalc=0, global_linear_attn_every=0)

Stacks n_layers EGNN_Sparse layers. Supports optional learnable token-embedding tables for integer-typed node and edge features, periodic edge-feature recalculation, and global linear attention every k layers.

Parameters

  • n_layers (int) – Number of EGNN_Sparse layers to stack.
  • feats_dim (int) – Node feature dimensionality (after optional embedding projection).
  • embedding_nums (list[int], optional) – Vocabulary sizes for each integer node feature dimension to embed. Default: None.
  • embedding_dims (list[int], optional) – Output embedding sizes corresponding to embedding_nums. Default: None.
  • edge_embedding_nums / edge_embedding_dims (list[int], optional) – Same as above but for edge features. Default: None.
  • recalc (int) – Recalculate edge features every recalc layers. 0 = disabled. Default: 0.
  • global_linear_attn_every (int) – Insert a global linear attention layer every k layers. 0 = disabled. Default: 0.
class torch_pharma.models.dynamics.egnn.EGNNDynamics(num_atom_types, h_hidden_dim=256, num_encoder_layers=9, condition_on_time=True, conditioning=(), self_condition=True, include_charges=False, **egnn_kwargs)

Complete EVD-compatible denoising backbone. Wraps EGNN_Sparse_Network with the full EVD input/output protocol: builds fully-connected masked edges, concatenates time and property context onto node embeddings, centralizes coordinates, runs EGNN, then projects outputs to coordinate velocity and scalar predictions.

Parameters

  • num_atom_types (int) – Must match EquivariantVariationalDiffusion.num_atom_types. Default: 5.
  • h_hidden_dim (int) – Width of node scalar hidden features inside EGNN. Default: 256.
  • num_encoder_layers (int) – Depth of the underlying EGNN_Sparse_Network. Default: 9.
  • condition_on_time (bool) – Appends the scalar timestep t as an extra feature per node. Default: True.
  • conditioning (list[str]) – List of property names (keys in the batch) to condition generation on. Default: ().
  • self_condition (bool) – If True, doubles input dims to accept the self-conditioning tensor xh_self_cond. Default: True.
  • include_charges (bool) – Append integer atomic charge features alongside atom types. Default: False.

Example

from torch_pharma.models.dynamics.egnn import EGNNDynamics

dynamics = EGNNDynamics(
    num_atom_types=5,
    h_hidden_dim=256,
    num_encoder_layers=9,
    conditioning=["alpha", "mu"],  # property-guided generation
)

Methods

forward(batch, t, xh_self_cond=None)

Run one denoising step. Builds edges, embeds features, conditions on time and context, runs EGNN, returns predicted coordinate velocity and scalar noise.

  • batch (torch_geometric.data.Batch) – Noisy molecular batch containing x (coords) and h (atom types).
  • t (Tensor) – Timestep, shape (batch_size,), values in [0, 1].
  • xh_self_cond (Tensor, optional) – Previous denoised prediction for self-conditioning, same shape as batch.xh. Default: None.

Return type Tensor — predicted noise, shape (N, 3 + num_atom_types + optional_charges)

class torch_pharma.models.dynamics.egnn.CoorsNorm(eps=1e-8, scale_init=1e-2)

Learnable scalar normalization for 3D direction vectors. Normalizes each vector to unit length then multiplies by a learned scalar scale parameter. Stabilizes coordinate updates inside EGNN_Sparse.

Parameters

  • eps (float) – Epsilon added to the norm to prevent division by zero. Default: 1e-8.
  • scale_init (float) – Initial value of the learned scale parameter. Default: 1e-2.
class torch_pharma.models.dynamics.gcpnet.GCP(input_dims: ScalarVector, output_dims: ScalarVector, nonlinearities=('silu','silu'), scalar_out_nonlinearity='silu', scalar_gate=0, vector_gate=True, frame_gate=False, sigma_frame_gate=False, feedforward_out=False, bottleneck=1, vector_residual=False, vector_frame_residual=False, ablate_frame_updates=False, ablate_scalars=False, ablate_vectors=False, scalarization_vectorization_output_dim=3)

Geometry-Complete Perceptron layer. Processes ScalarVector inputs by projecting vectors to scalars via norm, running a scalar MLP, gating vector outputs with scalar activations, and (unless ablated) integrating local SE(3) frame information to achieve geometry completeness.

Parameters

  • input_dims (ScalarVector) – A ScalarVector(s, v) giving the scalar and vector input channel counts.
  • output_dims (ScalarVector) – A ScalarVector(s, v) giving the scalar and vector output channel counts.
  • nonlinearities (Tuple[str, str]) – Activation functions for scalar and vector channels. Default: ('silu', 'silu').
  • vector_gate (bool) – Gate vector outputs with sigmoid of a scalar projection. Default: True.
  • frame_gate (bool) – Gate vectors using frame-derived directions (full geometric gating). Default: False.
  • sigma_frame_gate (bool) – Row-wise sigma gating using frame vectors. Default: False.
  • bottleneck (int) – Compression factor for the vector down-projection hidden dim. Default: 1.
  • ablate_frame_updates (bool) – Disable all frame-based updates. Reduces to GCP-Baseline. Default: False.

class torch_pharma.models.dynamics.gcpnet.GCP2(input_dims: ScalarVector, output_dims: ScalarVector, ...)

Improved GCP layer that fuses local frame scalarization directly into the scalar MLP input in a single forward pass. Produces direction-robust geometric features without the sequential update of GCP. Accepts the same constructor arguments as GCP.

class torch_pharma.models.dynamics.gcpnet.GCPEmbedding(edge_input_dims: ScalarVector, node_input_dims: ScalarVector, edge_hidden_dims: ScalarVector, node_hidden_dims: ScalarVector, num_atom_types: int, nonlinearities=('silu','silu'), pre_norm=True, use_gcp_norm=True, selected_GCP=GCP2, ...)

Initial embedding layer for GCPNet. Optionally looks up atom-type embeddings, applies GCPLayerNorm (pre or post), and runs separate GCP projections for node and edge ScalarVector representations.

  • node_input_dims (ScalarVector) – Scalar and vector input dims for nodes.
  • edge_hidden_dims (ScalarVector) – Target scalar and vector output dims for edges.
  • node_hidden_dims (ScalarVector) – Target scalar and vector output dims for nodes.
  • num_atom_types (int) – Number of atom types; if > 0 creates an nn.Embedding for atom-type lookup.
  • pre_norm (bool) – If True normalise inputs before embedding; otherwise normalise outputs. Default: True.
  • selected_GCP (type) – GCP variant class to use (GCP or GCP2). Default: GCP2.

class torch_pharma.models.dynamics.gcpnet.GCPMessagePassing(input_dims: ScalarVector, output_dims: ScalarVector, edge_dims: ScalarVector, reduce_function='sum', use_scalar_message_attention=True, use_residual_message_gcp=True)

Geometry-complete message passing. For each edge, constructs a message by concatenating sender, receiver, and edge ScalarVectors. Processes through a chain of GCP2 layers with optional residual connections (ResGCP). Applies learned scalar attention gating then aggregates messages by reduce_function.

Parameters

  • input_dims (ScalarVector) – Node scalar and vector input channel dims.
  • output_dims (ScalarVector) – Node scalar and vector output channel dims.
  • edge_dims (ScalarVector) – Edge scalar and vector channel dims.
  • reduce_function (str) – Aggregation function: 'sum', 'mean', or 'max'. Default: 'sum'.
  • use_scalar_message_attention (bool) – Learn a scalar gate applied to aggregated scalar messages. Default: True.
  • use_residual_message_gcp (bool) – Add residual connections inside the message GCP chain (ResGCP). Default: True.
class torch_pharma.models.dynamics.gcpnet.GCPInteractions(node_dims: ScalarVector, edge_dims: ScalarVector, cfg: DictConfig, layer_cfg: DictConfig, dropout=0.0, nonlinearities=None, update_node_positions=False)

Full GCPNet interaction block. Composes GCPMessagePassing with a configurable feedforward network, layer normalization (GCPLayerNorm), and dropout (GCPDropout). Optionally updates node positions using the aggregated vector-channel outputs.

Parameters

  • node_dims (ScalarVector) – Node scalar and vector hidden channel dims.
  • edge_dims (ScalarVector) – Edge scalar and vector hidden channel dims.
  • dropout (float) – Dropout probability applied after message passing and after feedforward. Default: 0.0.
  • update_node_positions (bool) – If True, compute a position update from aggregated vector features. Default: False.
