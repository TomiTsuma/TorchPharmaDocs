torch_pharma.models

Neural network models for molecular generation, property prediction, and geometric deep learning.

Diffusion

EquivariantVariationalDiffusionSE(3)-equivariant continuous-time diffusion model for joint 3D coordinate and atom-type generation.
PredefinedNoiseScheduleFixed analytic noise schedule (cosine or polynomial), maps t → γ(t).
GammaNetworkLearnable monotone noise schedule as a positive-weight MLP. Requires loss_type='vlb'.
PositiveLinearLinear layer with softplus-constrained positive weights. Building block of GammaNetwork.

Dynamics — EGNN

EGNNDynamicsEVD-compatible denoising backbone. Wraps EGNN_Sparse_Network with time/context conditioning and CoG-centred output projection.
EGNN_Sparse_NetworkStack of EGNN_Sparse layers with optional token embeddings and global linear attention.
EGNN_SparseSingle sparse EGNN message-passing layer with optional Fourier encoding, soft edge attention, and CoorsNorm.
CoorsNormLearnable scalar normalization for 3D direction vectors. Prevents coordinate drift.

Dynamics — GCPNet

GCPInteractionsFull GCPNet interaction block: message-passing + feedforward + norm + dropout.
GCPMessagePassingGeometry-complete message passing using chained GCP2 layers and scalar attention gating.
GCPEmbeddingInput embedding: atom-type lookup + GCPLayerNorm + GCP projections for nodes and edges.
GCP2Improved GCP that fuses frame scalarization into the scalar MLP in a single pass.
GCPGeometry-Complete Perceptron: scalar+vector inputs, local frame update in two sequential passes.

Distributions & Sampling

NumNodesDistributionHistogram prior over molecule sizes p(N). Wraps torch.distributions.Categorical.
PropertiesDistributionEmpirical property distribution conditioned on molecule size. Enables guided generation.
CategoricalDistributionFixed histogram distribution over discrete categories with KL divergence evaluation.
QueueFixed-length FIFO for maintaining a rolling window of scalar metric values.

Shared Primitives

GCPLayerNormLayer normalization for ScalarVector inputs. Applies separate norms to scalar and vector channels.
GCPDropoutDropout that consistently zeros both scalar and vector channels of a ScalarVector.
