class torch_pharma.models.transformers.NumNodesDistribution(histogram)

An nn.Module that wraps a torch.distributions.Categorical fitted to a dataset node-count histogram. Provides .sample() and .log_prob() for use in the diffusion generative process. Buffers move with .to(device).

Parameters

  • histogram (dict[int, int]) – Mapping {num_nodes: count} from the training dataset.

Example

from torch_pharma.models.transformers import NumNodesDistribution

histogram = {9: 3450, 17: 1200, 29: 800}
dist = NumNodesDistribution(histogram)

n = dist.sample(n_samples=64)       # Tensor (64,) long
lp = dist.log_prob(n)               # Tensor (64,) float

Methods

sample(n_samples)

Sample n_samples integers from the prior distribution p(N).

  • n_samples (int) – Number of samples to draw.

Return type Tensor — shape (n_samples,), dtype long

log_prob(batch_n_nodes)

Evaluate log p(N) for a batch of node counts.

  • batch_n_nodes (Tensor) – 1D long tensor of node counts.

Return type Tensor — log probabilities, same shape as input

class torch_pharma.models.transformers.PropertiesDistribution(dataloader, properties, device, num_bins=1000)

An empirical distribution over molecular property values conditioned on molecule size. Built from the training dataloader by binning each property into num_bins histogram bins per molecule size. Used for conditional generation: sample a context vector consistent with a target molecule size.

Parameters

  • dataloader (DataLoader) – Training dataloader used to build the histograms.
  • properties (list[str]) – List of property keys to model (must match batch dict keys).
  • device (torch.device) – Device for storing the histogram tensors.
  • num_bins (int) – Number of bins per property per molecule size. Default: 1000.

Methods

set_normalizer(normalizer)

Attach a mean/MAD normalizer dict (e.g. from compute_mean_mad) for property normalization.

  • normalizer (dict) – Output of compute_mean_mad().

Return type None

sample(num_nodes=19)

Sample a single context property vector for a molecule with num_nodes atoms.

  • num_nodes (int) – Number of atoms. Default: 19.

Return type Tensor — context vector of shape (len(properties),)

sample_batch(num_nodes)

Batch version of sample: draw one context vector per element of num_nodes.

  • num_nodes (Tensor) – 1D long tensor, one entry per molecule in the batch.

Return type Tensor — context matrix of shape (B, len(properties))

normalize_tensor(tensor, prop)

Z-score normalize property values using stored mean and MAD.

  • tensor (Tensor) – Raw property values.
  • prop (str) – Property name, used to look up mean/MAD in the normalizer.

Return type Tensor — normalized values

class torch_pharma.models.transformers.CategoricalDistribution(histogram, mapping)

A fixed discrete distribution over named categories (e.g. atom types). Primarily used to compare generated molecule type distributions against the training distribution via KL divergence.

Parameters

  • histogram (dict[str, int]) – Mapping {category_name: count} from the training dataset.
  • mapping (list[str]) – Ordered list of all category names defining the distribution alphabet.

Methods

kl_divergence(samples)

Estimate KL[q ∥ p] where q is the empirical distribution of samples and p is this prior.

  • samples (list[str]) – Generated category labels.

Return type float — KL divergence value

class torch_pharma.models.transformers.Queue(max_len=50)

Fixed-length FIFO queue backed by a Python list. Evicts the oldest entry when full. Used during training to maintain a rolling window of recent metric values.

Parameters

  • max_len (int) – Maximum number of entries before eviction. Default: 50.

Example

from torch_pharma.models.transformers import Queue

q = Queue(max_len=50)
q.add(0.32)
q.add(0.28)
print(q.mean(), q.std())

Methods

add(val)

Append val to the queue. Evicts the oldest item if at capacity.

Return type None

mean()

Compute the mean of all current entries.

Return type float

std()

Compute the standard deviation of all current entries.

Return type float

torch_pharma.models.transformers.compute_mean_mad(dataloaders, properties, dataset_name)

Compute per-property mean and Mean Absolute Deviation (MAD) from the training dataloader. Returns a nested dict suitable for passing to PropertiesDistribution.set_normalizer().

Parameters

  • dataloaders (dict[str, DataLoader]) – Dict with at least a 'train' key.
  • properties (list[str]) – Property keys to compute statistics for.
  • dataset_name (str) – Dataset identifier string (used for logging).

Return type

dict[str, dict[str, float]] — {prop: {'mean': ..., 'mad': ...}}

Example

from torch_pharma.models.transformers import compute_mean_mad

normalizer = compute_mean_mad(dataloaders, properties=["alpha", "mu"], dataset_name="qm9")
torch_pharma.models.transformers.log_grad_flow_lite(named_parameters, wandb_run)

Plot the mean absolute gradient value per non-bias layer as a line chart to WandB. Useful for detecting gradient vanishing. Is a no-op if wandb_run is None.

Parameters

  • wandb_run (wandb.Run or None) – Active WandB run. Pass None to disable logging.

Return type

None
